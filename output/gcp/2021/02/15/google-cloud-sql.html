프록시 서버 열기

  • 공식 문서 참고

로컬에서 Cloud SQL의 데이터베이스에 접근하기 위해 프록시 서버를 열어준다.

  • Cloud SQL 프록시 설치 파일명을 cloud_sql_proxy.exe로 바꾸고 Cloud SDK가 설치된 경로의 bin 폴더에 넣으면 따로 환경변수를 설정하지 않아도 터미널에서 사용할 수 있다. (귀찮)

  • 프록시 서버 열기

      cloud_sql_proxy -instances=<INSTANCE_CONNECTION_NAME>=tcp:3306

    INSTANCE_CONNECTION_NAME은 여기 적혀 있다.
